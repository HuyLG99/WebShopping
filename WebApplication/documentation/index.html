Introduction

This HTML e-commerce template is part of a package. Matching full PSD template (including more banners), stationery (including logo), newsletter templates, fully working OpenCart version and a Zen Cart version are available at themeforest and graphicriver.

This template was made in adobe Illustrator cs5, then exported to photoshop and coded in HTML5 / CSS3. 

This is only a HTML version of "things for cuties". It has no cart, contact or other functions included. If you need a fully working e-commerce website, you have to go further with this template and include it in your favorite shop software or you have to purchase the open cart / zencart version.

And dont forget: With your purchase you have supported the children's charity UNICEF. I donate 15% of every sale!

CSS Files

There are several CSS files that will be loaded at all pages.

  • "mobile.css" - as the name says, only for mobile phone and tablet displays like iOS or Android
  • "stylesheet.css" - contains all main styles, needed for all versions and all subpages
  • "carousel.css" - styles for the jquery carousel slider
  • "colorbox.css" - for the colorbox jquery module
  • "facebook.css" - some styles for the facebook box at the custom footer
  • "cloud-zoom.css" - styles for the cloud-zoom jquery module at the product page

Javascript Files

We load 4 javascript files at all pages. Beound this, the opencart system loads some more, i.e. the jquery library.

  • cloud-zoom.1.0.2.js - jquery image magnifier plugin; used on product pages
  • cuties.js - specific js code for this theme
  • jquery-1.7.1.min.js - the main jquery file
  • jquery.colorbox-min.js - the colorbox module, used for product image popup
  • jquery-cycle.all.js - jquery cycle plugin (with transition definitions); used for sliders
  • jquey.jcarousel.min.js - jquery carousel plugin for the brands and thumbs sliders
  • jquery.selectBox.js - a jquery plugin for beautyfull styled select boxes

Working with the PSDs

You can use the included elements PSDs (e.g. "product_box.psd" etc.), if you wanna change graphical elements for your page, but most of it are styled with CSS instead. If you like to change the whole design / layout, best way is to work directly with the PSD version of "things for cuties" - available here. At this point you definitly need more experience with Photoshop, Illustrator and HTML and I think, you do not need any help from my side... :)

So lets just change the image inside the "slider". Follow these easy steps:

  1. Open the "slider.psd" in Photoshop.
  2. As you can see at the layers window, there are only two layers inside. This is because the banner is a vector smart object, built in Illustrator. So double-click on the layer "big banner" now.

     
  3. The banner will open up at Illustrator now. Open the layers window now.
  4. Open the layers group "content layout 1" -> "big banner" -> "image".
  5. Inside this group you find the layer "image", which is just a gray rectangle.

     
  6. Place your favorite image now by using the Menu ("File" -> "Place").
  7. Move it inside the group "image".
  8. Resize it to fit inside the box.
  9. Delete the grey rectangle "image".
  10. You probably wanna change the text, so just use the type tool and do it.
  11. Save the vector smart object.
  12. Switch back to Photoshop.
  13. Use the menu "File" -> "Save for web and devices" at Photoshop.
  14. Save it as "png24" with transparency under a unique name inside the "images"-folder
  15. Now you have to change the slider image inside the html file.
